      Parameters

      • data: { latitude: number; longitude: number }[]

        座標資料陣列，每個元素包含 latitude 和 longitude

      Returns Promise<RTKDataModel[]>

      Promise<RTKDataModel[]> 成功建立的 RTK 資料模型陣列

      當資料格式錯誤或資料庫操作失敗時拋出錯誤

      const coordinates = [

        { latitude: 25.034123, longitude: 121.565789 }
