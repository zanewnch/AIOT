    Class UserController

    使用者管理控制器，處理系統使用者的CRUD操作

    提供使用者的創建、查詢、更新和刪除功能。 使用者是RBAC系統的基礎實體，與角色形成多對多關係。

    Example

    const userController = new UserController();
    app.use('/api/rbac/users', userController.router);
