    Class InitController

    初始化控制器，處理系統初始化相關的API請求

    提供系統演示資料的初始化功能，包括RBAC權限控制和RTK定位資料。 所有初始化操作都是冪等的，多次執行不會產生重複資料。

    Example

    const initController = new InitController();
    app.use('/api/', initController.router);
