    Class RBACController

    RBAC主控制器，整合所有角色基礎存取控制功能

    統一管理使用者、角色、權限以及它們之間的關聯關係。 所有RBAC相關的路由都會通過JWT驗證中間件進行保護。 使用依賴注入容器來管理各個子控制器的實例。

    Example

    const rbacController = new RBACController();
    app.use('/api/', rbacController.router);
