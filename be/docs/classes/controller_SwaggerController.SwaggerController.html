    Class SwaggerController

    Swagger文檔控制器

    提供API文檔的存取功能，包括JSON格式的OpenAPI規格和交互式UI介面。 支援前端整合和獨立的文檔瀏覽。

    Example

    const swaggerController = new SwaggerController();
    app.use('/api/', swaggerController.router);
