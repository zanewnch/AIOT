        斷開MongoDB連接

        安全地關閉與MongoDB的連接。只有在連接存在時才會執行斷開操作， 包含錯誤處理以確保斷開過程的穩定性。

        disconnectMongoDB

        Returns Promise<void>

        Example

        import { disconnectMongoDB } from './MongoDBConfig';

        // 在應用程式關閉時調用
        process.on('SIGINT', async () => {
          await disconnectMongoDB();
          process.exit(0);
        });
